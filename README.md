# aws-serverless-notes-app
A serverless notes web application built using AWS services such as Lambda, API Gateway, DynamoDB, and S3.

PROJECT DESCRIPTION

This project demonstrates how to build a cloud-based notes application using AWS serverless architecture. Users can create, view, and delete notes through a simple web interface.

The backend will be built using **AWS Lambda and API Gateway**, while notes will be stored in **Amazon DynamoDB**. The frontend will be hosted using **Amazon S3 static website hosting**.

AWS SERVICES USED:

* AWS Lambda
* API Gateway
* Amazon DynamoDB
* Amazon S3
* CloudWatch
